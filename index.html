Product Scope

Scope In

  • Capture Screen from browser and import page context
  • Prompt-driven normalization into a chosen output schema
  • Create and update cards on a board
  • Automated board rules for status, due-date, and stale-task control

Scope Out

  • Not a full CRM/ATS/ERP replacement
  • Not an OCR-only utility with no workflow automation
  • Not a "zero-config autopilot" product

How It Works

1 Capture screen or pull text context from the active page.
2 Apply a prompt plus output schema (JSON/Markdown/CSV) to structure the action payload.
3 Create a card on the board because the board is the execution surface, not storage only.
4 Board automation rules run automatically to control statuses, deadlines, stale tasks, and follow-ups.
